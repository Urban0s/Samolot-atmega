# Projekt Samolotu Zdalnie Sterowanego

Projekt modelu samolotu RC zbudowanego ze styroduru, którego sercem jest mikrokontroler **ATmega328P**. Oprogramowanie sterujące zostało napisane od zera w języku C, a cała konstrukcja została zaprojektowana z myślą o lekkich i wytrzymałych materiałach modelarskich.

---

## 🛠️ Specyfikacja techniczna i komponenty
* **Główny kontroler:** Mikrokontroler ATmega328P (taktowany zewnętrznym kwarcem)
* **Napęd:** Silnik bezszczotkowy (BLDC) + regulator obrotów ESC
* **Sterowanie:** Serwomechanizmy SG90 odpowiedzialne za powierzchnie sterowe
* **Konstrukcja:** Kadłub i skrzydła ze styroduru, wzmocnione dźwigarami z rurek węglowych, całość pokryta folią termokurczliwą
