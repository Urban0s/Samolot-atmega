# Projekt Samolotu Zdalnie Sterowanego

Projekt modelu samolotu zbudowanego ze styroduru, którego sercem jest mikrokontroler **ATmega328P**. Oprogramowanie sterujące zostało napisane od zera w języku C, a cała konstrukcja została zaprojektowana z myślą o lekkich i wytrzymałych materiałach modelarskich.
