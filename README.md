# Projekt Samolotu Zdalnie Sterowanego

Projekt zdalnie sterowanego modelu samolotu wykonanego własnoręcznie ze styroduru. Głównym układem sterowania jest mikrokontroler **ATmega328P**, dla którego oprogramowanie sterujące zostało napisane w czystym języku C. Projekt łączy w sobie programowanie niskopoziomowe, elektronikę oraz modelarstwo lotnicze.
